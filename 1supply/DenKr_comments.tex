%        3: The Color to use
%    - Created Macros:
%        "Tagged Comment" -> \<Abbr>{}: Prints text, suffixed with "#<Abbr>: " (and colored)
%        "Highlighting" -> \<Abbr>hl{}: Just prints the text in Color.
%    -> E.g.: \newcommandsDisw{dekr}{DenKr}{<Color>} creates the two Macros

%        \dekrhl{}
